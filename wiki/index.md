## Retrieval Guidance

- Prefer the narrowest page that directly matches the question.
- Use [[Claude Code Operating Patterns]] for workflow, commands, context, hooks, and team structure.
- Use [[Agent Skill System Design]] for skills, hooks, and reusable agent architecture.
- Use [[Open Model Runtime Selection]] for local-vs-API model choices, especially Gemma and Qwen.
- If the question is about durable repo policy, start from [[Notion Source Of Truth]] and then jump to the more specific page.

## Current Biases

- The source corpus is strong on Claude Code, agent workflows, hook-based quality gates, MCP token management, and open-model runtime decisions.
- The source corpus is still weak on product scoping outside AI tooling.
- Product-specific pages should be added when the same project theme comes up more than once.
